docker 基础命令使用

发表于 2020-08-11 分类于 docker 阅读次数： Valine：

docker命令

search

用于搜索docker registory中镜像信息

其中第一个（没有前缀的称为顶级/顶层仓库）,一般是docker官方的仓库。后面xxxx/nginx表示用户仓库

默认安装的镜像大部分用的centos/ubuntu，如果需要自定义，大部分都是自己制作镜像。

alpine版本：专门用于制作的小镜像
stable版本: 稳定版本镜像
latest: 最新版本镜像

pull

pull 用于下载镜像

images

列出本地下载的镜像，该命令和docker image ls 效果一致。

“ “ 这种镜像在Docker官方文档中被称作dangling images，指的是没有标签并且没有被容器使用的镜像。

rm

删除一个容器

rmi

删除一个镜像

删除

ps (container ls)

列出容器

start

启动一个容器

restart

重启一个容器

stop（-15）/kill（-9）

stop 停止一个或者多个容器
kill 强制停止一个或多个容器(不建议生产使用)

run

创建并启动一个容器run = create+start

-d: daemon 用于支撑容器

create

创建一个容器但是不启动它

创建一个ngx2的容器

version

查看docker版本
